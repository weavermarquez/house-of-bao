** Note that 

I am distinguishing between possible / meaningful because Enfolds and Creates with no nodes nor parents selected are created at the top-level and are trivially dismissable. Such a move is very likely to be a meaningless move to be undone.

It is notable that only Enfold and Create (and Sandbox for Sandbox mode), whose point at which they are possible != point at which they become meaningful are the only ones with this behaviour, and the only ones that care about Parents. The other four are not made any more meaningful by the presence of a Parent selection.

Additionally, actions do not become /more/ meaningful past the point when they become meaningful.
I wonder if this would be worth mapping the above out, or using it to determine the semantics for the radial menu. I would prefer not to have keyboard shortcuts. I don't want to deviate too far from the mobile focus.
